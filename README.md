## Get Started

Install dependencies:

```sh
npm install

yarn
```

Development:

```sh
npm run dev

yarn dev
```

Build:

```sh
npm run build

yarn build
```

## About

- HUBusca is a solution for make easy to search a user on Github
- Is a website made with Next.js
